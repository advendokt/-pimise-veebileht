Õpi CSS-i Põhitõdesid

Mis on CSS?

CSS (Cascading Style Sheets) on stiililehtede keel, mida kasutatakse HTML-i elementide kujundamiseks ja paigutuse kontrollimiseks veebilehel. See võimaldab kohandada teksti, pilte ja muud sisu kujundust.

CSS-i selektorid

Valijad on CSS-is kasutatavad märgendid, mis määravad, millistele HTML-elementidele stiile rakendada:

  • Elementi valija (nt `p`, `h1`)
  • Klassi valija (nt `.klassinimi`)
  • ID valija (nt `#idnimi`)
  • Atribuudi valija (nt `[type="text"]`)

Levinud CSS-i omadused

Õpi tundma mõningaid olulisi CSS-i omadusi veebilehtede kujundamiseks:

  • Teksti omadused (nt `font-family`, `font-size`)
  • Värvuse omadused (nt `color`, `background-color`)
  • Kasti mudel (nt `margin`, `padding`, `border`)
  • Positsioneerimine (nt `position`, `top`, `left`)
  • Flexbox ja Grid paigutus
  • Üleminekud ja animatsioonid
