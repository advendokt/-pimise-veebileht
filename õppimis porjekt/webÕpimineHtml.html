HTML Elemendid ja Struktuur

HTML Elemendid

HTML (HyperText Markup Language) kasutab erinevaid elemente veebilehtede struktureerimiseks ja sisu määratlemiseks.

Siin on mõned olulised HTML elemendid:

  • <h1>, <h2>, <h3> - Pealkirjad
  • <p> - Lõigud (paragrahvid)
  • <a> - Lingid (ankurid)
  • <img> - Pildid
  • <ul>, <ol>, <li> - Loendid
  • <table>, <tr>, <td> - Tabelid

HTML Dokumendi Struktuur

HTML dokumendi struktuur koosneb mitmest olulisest elemendist:

  • <!DOCTYPE html> - Dokumendi tüübi deklaratsioon
  • <html> - HTML dokumendi juurelement
  • <head> - Metaandmed (nt pealkiri, meta kirjeldused)
  • <title> - Dokumendi pealkiri, mis kuvatakse brauseri vahekaardil
  • <body> - Dokumendi keha, kus asub sisu
