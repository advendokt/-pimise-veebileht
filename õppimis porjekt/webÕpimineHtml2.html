HTML Atribuudid ja Lingid

HTML Atribuudid

Atribuudid annavad lisateavet HTML elementide kohta ja neid määratakse elementidele avamisjärgus.

Näide HTML elemendist koos atribuudiga millega saab lisada pilte:

<img src="pilt.jpg" alt="Pilt">

HTML Lingid (Ankurid)

HTML-i lingid (<a>) võimaldavad luua hüperlingid teistele veebilehtedele või dokumentidele.

Näide HTML lingist:

<a href="https://www.example.com">Külasta Example.com lehte</a>
