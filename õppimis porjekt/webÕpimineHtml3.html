HTML Tabelid ja Vormid

HTML Tabelid

HTML-i tabelid (<table>) võimaldavad struktureerida andmeid ridade ja veergudena.

Näide HTML tabelist:

<table>
    <tr>
        <td>Nimi</td>
        <td>Vanus</td>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
    </tr>
</table>

HTML Vormid

HTML-i vormid (<form>) võimaldavad kasutajatel sisestada andmeid ja neid serverisse saata.

Näide HTML vormist:

<form action="/submit_form" method="post">
    <label for="name">Nimi:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Saada">
</form>
