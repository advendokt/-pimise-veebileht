JavaScripti Funktsioonid

Töötamine Funktsioonidega

Funktsioon on plokk korduvkasutatavat koodi, mis teostab konkreetset ülesannet. JavaScriptis määratakse funktsioonid kasutades function võtmesõna.

Funktsioonid võivad võtta parameetreid (sisendväärtusi) ja tagastada tulemusi. Need on olulised koodi organiseerimiseks ja modulariseerimiseks, soodustades korduvkasutust ja hooldatavust.

Näide

// Funktsioon tervitussõnumi genereerimiseks
function tervita(nimi) {
return "Tere, " + nimi + "!";
}
// Funktsioon, mis käivitatakse nupu klõpsamisel
function kuvaTervitus() {
const nimi = document.getElementById("nimi").value;
const tervitus = tervita(nimi);
document.getElementById("tervitus").textContent = tervitus;
}

Funktsioon tervita võtab parameetriks nimi ja tagastab tervitussõnumi, mis sisaldab antud nime.
